# DayBreak Enterprise

A full business management suite built with React 19, TypeScript, Convex, and Tailwind CSS 4.

**Live app:** https://daybreak-enterprise.onhercules.app/

## Features

- **Dashboard** — Revenue tracking, outstanding invoices, quick stats
- **Clients** — Full client directory with contact details
- **Products & Services** — Product catalog with pricing
- **Quotations** — Create and send professional quotes
- **Invoices** — Generate invoices and track payments
- **Receipts** — Issue receipts after payment
- **Settings** — Business profile and default currency (ZMW default)
- **Light / Dark / System** theme toggle in sidebar
